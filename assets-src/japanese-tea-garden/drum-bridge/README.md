# Drum Bridge texture sources

These two source images were generated on 2026-07-13 with Codex's built-in
image-generation mode. The Japanese Tea Garden reference photos were used only
for material, color, and weathering reference. The generated sources are kept
here; `npm run build:drum-bridge-textures` mirror-wraps them, derives tangent
space normals from the authored grain, and emits the runtime KTX2/WebP pairs in
`public/japanese-tea-garden/drum-bridge/`.

## Painted timber prompt

Create a perfectly seamless square tile of weathered painted timber matching
the historic Drum Bridge. The full image is a flat orthographic material
swatch: close-grained cypress or cedar beneath muted red-brown paint, visible
longitudinal grain, subtle brush direction, restrained sun fading, fine
hairline cracks, narrow wear, and tiny darker damp flecks. Use diffuse neutral
albedo lighting with no cast shadows, highlights, ambient occlusion, plank
gaps, nails, architecture, plants, water, text, watermark, frame, perspective,
vignette, black crevices, orange saturation, or obvious repeated motifs. Grain
runs horizontally and all four edges tile seamlessly.

## Worn tread prompt

Create a perfectly seamless square tile of old foot-worn timber matching the
gray-brown stair boards on the historic Drum Bridge. The full image is a flat
orthographic material swatch: tightly grained cypress or cedar with most of the
red-brown coating rubbed away, silvery warm-gray wood, subtle pigment remaining
in grain lines, fine shallow checks, softly rubbed wear, and tiny damp specks.
Use diffuse neutral albedo lighting with no cast shadows, highlights, ambient
occlusion, plank gaps, nails, architecture, plants, water, text, watermark,
frame, perspective, vignette, black crevices, or obvious repeated motifs. Grain
runs horizontally and all four edges tile seamlessly.
